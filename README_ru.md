# YouTube Feed Cleaner

Элегантное и легковесное расширение для Google Chrome, предназначенное для тонкой настройки и очистки вашей ленты YouTube. Позволяет сфокусироваться на интересном контенте, автоматически скрывая навязчивые элементы и уже просмотренные видео.

## Особенности

Расширение предоставляет удобный пользовательский интерфейс (в стилистике темной темы Obsidian с красными акцентами) для управления следующими функциями:

* **Скрытие «Джемов» (Mixes):** Убирает автогенерируемые бесконечные плейлисты из общей ленты и рекомендаций.
* **Блокировка Shorts:** Позволяет раздельно отключать показ коротких видео на Главной странице и в результатах Поиска.
* **Фильтрация просмотренного:** Автоматически скрывает карточки видео на Главной странице, если прогресс их просмотра составляет 70% и более.
* **Фильтрация по дате публикации:** Позволяет скрыть с Главной страницы видеоролики, которые старше заданного вами периода (от 1 недели до 1 года).
* **Мультиязычность:** Интерфейс расширения автоматически адаптируется под язык вашего браузера (встроенная поддержка русского и английского языков).
